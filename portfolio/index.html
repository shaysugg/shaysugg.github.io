FlexiTrace

App Store / Website

FlexiTrace Posture and Gait Analysis app helps Biomechanics and Sport Experts easily evaluate their clients with real-time and on-device AI-powered tools such as; Markerless Gait analysis, Posture Analysis Tool, Real-time joint measurement and movement analysis, and a library of corrective exercise plans with muscle animated videos.

ComicKhan

GitHub

ComicKhan is an app that extracts comic files (.cbz /.cbr) images, attach metadata like name, group, last read page and organizes them in a comic book library. It automatically detects new comic files that are added into the app’s document directory and starts extracting them on background while showing the extraction progress to the user. It performs several resizing operations on comic-book images for better performance. The comic book reader has an adaptive layer for reading in single/double page mode, light/dark background, and more.

My Role

  • Performed research, designed the UI, and developed the app from scratch.
  • Used CoreData to design the metadata database and FileManager to persist comic images.
  • Combined and customize several UI components to create a comic book reader.
  • Took full ownerships of the app and maintained it on AppStore for 2 years.

Technologies

Swift, UIKit, CoreData, GCD, FileManager, Combine, Zip, UnrarKit, CoreGraphics
